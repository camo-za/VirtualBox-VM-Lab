# 🧰 VirtualBox VM Lab

This project demonstrates the setup and configuration of both **Windows 10/11** and **Ubuntu 22.04 LTS** virtual machines using **Oracle VirtualBox**.  
It serves as part of my **CompTIA A+ certification study** and my journey toward an **IT Support / SysAdmin** role.

---

## 🖥️ Project Overview
This lab was designed to simulate real-world IT support and system administration tasks in a safe, virtual environment.  
It covers installation, configuration, snapshot management, and basic troubleshooting for multiple operating systems.

---

## ⚙️ Virtual Machines Created

### 🪟 Windows VM
- Installed from Windows 10/11 evaluation ISO  
- Configured system restore point and user accounts  
- Installed essential tools (7-Zip, Chrome, Notepad++)  
- Practiced Windows Update, Device Manager, and Task Manager usage  
- Screenshots:  
  - 🖼️ *Installation Complete*  
  - 🖼️ *System Restore Point Created*  
  - 🖼️ *VirtualBox VM Settings*

### 🐧 Linux VM (Ubuntu 22.04 LTS)
- Installed via Ubuntu ISO  
- Configured static IP address and SSH access  
- Installed Nginx web server  
- Set up UFW firewall rules and basic user permissions  
- Screenshots:  
  - 🖼️ *Ubuntu Desktop View*  
  - 🖼️ *Nginx Welcome Page*  
  - 🖼️ *UFW Firewall Enabled*
